🚀 ЯК ЗАПУСТИТИ:

Варіант 1: Подвійний клік на файл
   → Відкрийте index.html у браузері

Варіант 2: Використати батник (рекомендовано)
   → Подвійний клік на start-server.bat
   → Відкриється http://localhost:8000

Варіант 3: Через командний рядок
   PowerShell:  python -m http.server 8000
   або:         npx http-server -p 8000

🌐 РОЗГОРТАННЯ:

📤 На хостинг:
   1. Завантажте всі файли через FTP
   2. Налаштуйте домен prodiesel.com.ua
   3. Увімкніть HTTPS (SSL-сертифікат)

   1. Створіть репозиторій
   2. git init && git add . && git commit -m "Initial"
   3. git push origin main
   4. Settings → Pages → Enable
